== Description ==

Provides a simple way to manage items in and around the footer of a Thesis Theme.

= Features =

* Remove or replace the Thesis Attribution line with your affiliate link.
* Add text or code above, inside, and below the footer.

== Installation ==

1. Upload `thesis-footer-tool` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin on the settings screen.
